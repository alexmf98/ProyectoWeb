## Sobre Construcción Sarmiento

Construcciones Sarmiento es una aplicación web diseñada para facilitar la comunicación entre la empresa y sus clientes, trabajadores y entidades públicas. Permite gestionar proyectos de construcción de forma centralizada, eliminando la necesidad de comunicación constante gracias a funcionalidades como la visualización de imágenes en tiempo real, la gestión documental para trabajadores y el seguimiento del estado de cada proyecto.
Además, incorpora un sistema de alquiler de maquinaria de construcción que permite a los usuarios consultar la disponibilidad, realizar reservas y gestionar sus alquileres de forma sencilla y controlada:
